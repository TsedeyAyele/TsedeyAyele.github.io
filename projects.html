Mobile Repair App

This project aims to develop a Mobile Repair App to manage customer information, mobile device details and repair jobs. The app is developed in Java and uses MySQL for data storage. Furthermore, it leverages Amazon RDS for database hosting. Mobile Repair App provides functionalities such as adding, updating, viewing, and deleting customers, mobile devices, and repair jobs.
